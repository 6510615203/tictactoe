# TicTacToe
เป็นเกม XO โดยหากฝั่งไหนเลือกใส่ได้ครบสามช่องในแนวนอน แนวตั้ง หรือแนวทแยงได่ก่อนฝั่งนั้นจะเป็นฝ่ายชนะ โดยในเกมนี้ X คือคอมพิวเตอร์ และ O คือผู้เล่น จะสลับกันเริ่มเล่นไปเรื่อยๆ
